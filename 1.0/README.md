memo
====

__version 1.0__

## 介绍

一个轻量的便签软件

## 支持平台说明

__基本运行环境__
python2.7 pyqt4

__window linux 均可运行__
_附加说明_
1. windows下，程序支持热键 `\``,（需要从hotkey.py运行程序）
2. 也可以直接运行widget.py （不附带热键功能）

## 项目说明
- `exe.py` 支持windows下，把程序打包成exe
- `memo.sh` 支持bash，sh运行
